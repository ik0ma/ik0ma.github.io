misc基础(一)

个人感觉杂项就是要多接触一些知识点,积累的多了根据题目的提示可以直接了解做题的方法

0x01 一些文件文件头格式.

  • JPG格式

    根据上图可以知道.jpg更改图像的宽,高,颜色.

  • PNG格式

  • ZIP格式

    0X 50 4b 03 04

分文件头信息标志，一般是zip文件的开头，可以通过这个判断文件格式.

0x02 编码 算法 密码类

绝大部分题目都会涉及到这些问题,最好多多了解(做笔记)
下面我每种类型都写几个例子类型

  • Base 64 32 16 (特征：密文字符串的长度为4的整数倍。字符串的符号取值只能在A-Z, a-z, 0-9, +, /, =共计65个字符中，且=如果出现就必须在结尾出现。)
  • MD5加密 (特征:MD5密文一般为16位或32位不规则中英文混合字符串)
  • 摩斯电码 (特征:只有 - 和 *)
  • 培根密码 (特征:只出现 a 和 b)
  • 栅栏密码 (特征:栅栏密码没有变换字符的内容，所以仍有元音保留，它的字符出现频率自然接近标准。)
  • 凯撒密码 (特征:多数情况会把元音变成非元音，非元音大多数依然变成非元音，因此在经过这种加密算法后，元音字母会很少。)
  • unicode (形如: \u0059\u006f\u0075\u0020\ )
  • ASCII编码 (ASCII详解)
    …
    还有很多没有接触的crypty类型还需要去多多了解,等了解了再加上.
